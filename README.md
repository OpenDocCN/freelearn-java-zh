## 下载

### Docker

```
docker pull apachecn0/flygon-java-zh
docker run -tid -p <port>:80 apachecn0/flygon-java-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g flygon-java-zh
flygon-java-zh <port>
# 访问 http://localhost:{port} 查看文档
```
