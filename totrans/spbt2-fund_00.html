Preface

The Spring Framework is a mature, enterprise-grade, and open source application development framework that provides a flexible and modern alternative to the official Java EE standard.

Who This Book Is For

This book is aimed at developers who are at the beginner or early-intermediate stage of their career and want to be able to easily create robust web applications or RESTful services using the Java platform. You should have at least basic Java knowledge and know how to compile an application with a given POM file with Maven. It is assumed that you are at the beginner or early intermediate stage. You don't need to be an HTML expert but should know how HTML works and how to keep a file XML/XHTML compliant.

You should take this course if you want to create modern web applications or RESTful services with Java. You will also learn a lot about the Spring Framework itself, which will enable you to create complete applications on your own afterward.

What This Book Covers

Chapter 1, The Spring Project and Framework, introduces us to the Spring Framework and its principles. Then we get our first taste of constructing and running projects. After focusing on the main building blocks of Spring, this chapter finally shows us how to use the Project Lombok library.

Chapter 2, Building a Spring Application, takes us through the configuration classes of Spring Beans, as well as their various dependencies, before showing us exactly how to create and configure different environments.

Chapter 3, Testing Spring Applications, covers the creation and analysis of the two different types of tests, namely unit and integration tests.

Chapter 4, The MVC Pattern, talks about the aforementioned pattern, explaining the model, view, and the controller in detail. The chapter briefs us on the development of applications and implementation of the different controllers.  

Chapter 5, Displaying Information Using Web Pages, trains the students on the template engine Thymleaf, it's syntax; and elements; then how to build web pages using Thymleaf.  

Chapter 6, Passing Data Between the View and the Controller, teaches us about the creation of forms and different input fields used when entering information in a web browser.

Chapter 7, RESTful APIs, covers the different aspects of REST, how to use Postman, and the process of writing RESTful APIs.

Chapter 8, Web Application Security, discusses the aspects of web app security, and the different security options in Spring.

Chapter 9, Persisting Data Using a Database, covers choosing a database management system and working with relational databases and SQL;  understanding database accessing using JDBC and JdbcTemplate, as well as further data implementation using Spring.

To Get the Most out of This Book

To complete this book successfully, you will require computer systems with at least an Intel Core i5 processor or equivalent, 8 GB RAM, and 5 GB available storage space.  Along with this, you would require the following software:

  • Operating system: Windows 7 or higher
  • Browser: Google Chrome or Mozilla Firefox (latest updates installed)
  • IntelliJ Community Edition (or Ultimate) – the latest version
  • JDK 8+
  • Maven 3.3+
