本部分包含以下章节：

+   *第四章*, *异常处理*

+   *第五章*, *字符串、输入/输出和文件*

+   *第六章*, *数据结构、泛型和常用工具*

+   *第七章*, *Java 标准和外部库*

+   *第八章*, *多线程与并发处理*

+   *第九章*, *JVM 结构和垃圾回收*

+   *第十章*, *数据库中的数据管理*

+   *第十一章*, *网络编程*

+   *第十二章*, *Java GUI 编程*
